# 💼 Bangalore Tech Salary Decoder

## 📌 Overview

This repository contains my solution for the **Bangalore Tech Salary Decoder**, a live industry project conducted by **Unlox Academy**.

The objective of this project was to analyze a real-world inspired dataset of Bengaluru tech professionals using **Python, Pandas, and NumPy** to uncover meaningful salary trends based on job roles, experience, skills, company type, and education.

---

## 🎯 Project Objectives

* Load and inspect the dataset
* Clean and preprocess inconsistent data
* Standardize salary formats and categorical values
* Perform Exploratory Data Analysis (EDA)
* Analyze salary distribution across different job roles
* Study salary growth with experience
* Compare salary premiums for technical skills
* Analyze company-type salary differences
* Identify underpaid professionals
* Generate a structured Salary Intelligence Report

---

## 📊 Dataset

* **Dataset Name:** `bangalore_tech_salaries.csv`
* **Records:** 1,000+ Bengaluru Tech Professionals
* **Year:** 2024 (Synthetic Industry Dataset)

### Features
